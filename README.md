## Идея

**Микросервис** — это небольшой независимый модуль, выполняющий одну чётко очерченную функцию и взаимодействующий с другими через простые интерфейсы (философия Unix: «делай одну вещь и делай её хорошо»). Легко устанавливается и обновляется, может быть реализован на разных языках.

## Что предлагаем

- **Готовые микросервисы** — магазин решений, готовых к установке.  
- **Разработка своего микросервиса** — под ваши процессы и интеграции.  
- **Разработка в складчину** — предзаказ и совместное финансирование востребованных модулей.

## Направления

- **Расширения для 1С** — доработки без снятия с поддержки, интеграции по API, построение своей экосистемы.  
- **Чат-боты Telegram** — сценарные диалоги, рассылки, приём заявок с записью в 1С.  
- **AI-сервисы** — контент для сайтов/маркетплейсов, обработка отзывов и вопросов, ответы ботов.  
- **RPA-роботы** — эмуляция действий в браузере, парсинг, цепочки «почта → файлы → 1С и т.д.».
